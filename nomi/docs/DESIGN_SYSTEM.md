# NOVA Design System

## Colors
Defined in `public/style.css`:
- Primary: `#3FB292`
- Primary Dark: `#31997b`
- Headline: `#183939`
- Body: `#4B5563`
- Background: `#E6E9ED`

## Typography
- Base font: `Aptos` with system fallbacks
- Headline weight: 700
- Body weight: 400-500

## Spacing
- Card padding: 28px
- Button padding: 12px 22px
- Container width: `min(1100px, 92%)`

## Components
- Cards: `.card`
- Navbar: `.navbar`
- Buttons: `.button` or `button`
- Skeletons: `Skeleton`, `SkeletonCard`, `SkeletonList`

## Accessibility
- Focus outlines for interactive elements
- Skip-to-main link
- Minimum touch target size 44x44px

## Animations
- Subtle hover lift on cards
- Fade-in for welcome overlay
